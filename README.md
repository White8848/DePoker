# DePoker - 德扑记账应用 🎮

基于区块链技术的德州扑克记账应用，确保游戏过程的透明性和数据不可篡改性。

## 快速开始

1. 安装依赖

   ```bash
   npm install
   ```

2. 启动应用

   ```bash
   npm start
   ```

3. 运行应用

   - Android: `npm run android`
   - iOS: `npm run ios`
   - Web: `npm run web`

## 功能特性

✅ 创建游戏房间  
✅ 玩家买入管理  
✅ 实时筹码追踪  
✅ 盈亏自动计算  
✅ 游戏结算与转账指引  
🔄 区块链验证（开发中）
